# Dev-network

You can create your resume, portfolio in a simple way and share with others

## Built With

- Frontend
  - React, Redux, ES6, React Hooks
- Backend

  - MongoDB, Nodejs, Express

## Getting Started

```
git clone https://github.com/jiminyun/dev-profile
npm install

npm run dev
```
